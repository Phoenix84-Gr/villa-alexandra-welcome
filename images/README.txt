!!  WINDOWS DOUBLE-EXTENSION TRAP — READ THIS  !!

When you type "villa-hero.jpg" in the Save As dialog, Windows often
appends ".jpg" automatically — saving the file as "villa-hero.jpg.jpg".
The page won't find it because the HTML expects "villa-hero.jpg" exactly.

FIX:
- In File Explorer: View > tick "File name extensions" (so you can see
  the real filename), then rename and delete the duplicate ".jpg".
- Or in the Save As dialog, change "Save as type" to "All Files"
  BEFORE typing the filename.
